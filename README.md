🏠 Real Estate Market Analysis Dashboard (India 2025)
📌 Project Overview

This project analyzes the Indian real estate market using Power BI. The dashboard provides insights into property prices, market trends, property types, city-wise performance and affordability segments to support data-driven decision-making for buyers, investors and real estate businesses.

🎯 Objectives
- Analyze property prices across major Indian cities.
- Identify premium and affordable housing markets.
- Track monthly and daily price trends.
- Compare property type distributions.
- Evaluate price-per-square-foot variations.
- Generate actionable insights for stakeholders.

🛠 Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Cleaning & Transformation
- Data Visualization

📊 Dashboard Features

Executive Summary
- Total Properties Listed
- Average Property Price
- Average Price per Sq.ft
- Number of Cities Covered

Market Trend Analysis
- Daily Price Trends
- Monthly Price Trends
- Growth Patterns

City-wise Analysis
- Property Distribution by City
- Average Property Price by City
- Price per Sq.ft Comparison

Property Type Analysis
- Apartments
- Villas
- Independent Houses
- Other Property Categories

Affordability Segmentation
- Affordable Properties
- Mid-Range Properties
- Premium Properties

🔍 Key Insights
- Identified cities with the highest property appreciation.
- Compared premium and affordable housing markets.
- Analyzed demand distribution across property types.
- Evaluated city-level price-per-square-foot variations.
